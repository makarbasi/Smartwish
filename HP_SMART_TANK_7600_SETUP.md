## ğŸ“ Paper Loading Instructions

### Recommended Paper

- **Type**: Heavyweight Cardstock
- **Weight**: 200-300 GSM (grams per square meter)
- **Size**: Letter (8.5 Ã— 11 inches)
- **Finish**: Matte or Semi-Gloss
- **Brand**: HP Premium Plus Photo Paper or similar

### Loading the Paper

1. **Use Main Paper Tray** (better for thick cardstock)
2. **Orientation**: Place paper in LANDSCAPE
   - 11 inch edge at the top (feeding edge)
   - 8.5 inch edge on sides
3. **Adjust Paper Guides**:
   - Slide width guides to fit Letter size exactly
   - Don't leave gaps or make too tight
4. **Load Only 10-20 Sheets**:
   - Thick cardstock needs space
   - Prevents jams

## ğŸ”§ Troubleshooting

### Problem: White Borders Showing

**Solution:**
- Ensure "Borderless" is enabled
- Check margins are set to 0" or None
- Try "Fit to Page" or "Full Bleed" option
- Verify paper size matches exactly (Letter)

### Problem: Wrong Side Orientation

**Solution:**
- Check duplex flip is set to SHORT EDGE
- NOT Long Edge (that's for portrait)
- Verify landscape orientation is selected

### Problem: Paper Jams

**Solution:**
- Use main paper tray (not rear feeder)
- Don't overload - only 10-20 thick sheets
- Adjust paper guides to fit snugly
- Ensure paper is properly aligned

### Problem: Low Quality Print

**Solution:**
- Set print quality to BEST/High (not Draft)
- Verify 300 DPI or higher is selected
- Use proper cardstock (200-300 GSM)
- Check ink levels in Smart Tank

### Problem: Colors Look Wrong

**Solution:**
- Verify Color mode is selected (not Grayscale)
- Check paper type is set to Heavyweight Matte
- Run HP printer color calibration
- Ensure ink tanks are full
